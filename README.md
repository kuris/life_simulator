# 🏢 대한민국 직장인 생존기

> **내가 직접 살아보는 근현대사** — 출근부터 퇴근까지, 선택이 인생을 바꾼다

---

## 🎮 게임 소개

대한민국 1980년대 ~ 2026년까지 6개 시대를 배경으로 한 **텍스트 기반 머드게임 스타일 직장인 생존 시뮬레이터**.  
캐릭터를 만들고, 하루를 살아내며, 다양한 선택지로 결말이 달라집니다.

---

## ✅ 구현된 기능

### 🕰️ 시대 선택 시스템 (6개 시대)
| 시대 | 배경 | 특징 |
|------|------|------|
| 1980년대 | 고도성장기 | 야근 기본, 낮은 물가, 체력 소모 높음 |
| 1997 IMF | 외환위기 | 구조조정, 최고 난이도, 역사적 이벤트 |
| 2000년대 | IT 붐 | 벤처 창업, 닷컴 버블 |
| 2010년대 | 스마트폰 | 카톡 야근, 유튜버 등장 |
| 2020 코로나 | 팬데믹 | 재택근무, 자영업 위기 |
| 2026 AI시대 | 현재 | AI 대체, 플랫폼 노동 (첫 플레이 추천) |

### 🃏 시대 카드 UI (신규)
- **한 줄 운명**: 각 시대의 핵심을 한 문장으로 ("버티면 집을 살 수 있는 시대")
- **난이도 시각화**: ★☆ 별점 + 색상 코딩 (빨강=위험, 노랑=보통, 녹색=쉬움)
- **추천 뱃지**: 2026년대에 🟩 첫 플레이 추천 표시
- **클릭 프리뷰**: 카드 클릭 시 상세 정보 & "당신은 이런 인생을 살게 됩니다" 펼침
- **시작 버튼**: 프리뷰 열린 후 확인 버튼으로 시작

### 👤 캐릭터 생성 (5단계)
1. 고용 형태 선택 (정규직/계약직/알바/프리랜서/자영업)
2. 직종 선택 (시대별 6종 직업, 총 36종)
3. 배우자 유무
4. 자녀 유무 (배우자 있을 때만)
5. 근무지 + 거주지 선택 (통근거리 영향)

### ⚡ 역사적 이벤트 (시대별 1회)
- IMF 외환위기 발표 → 구조조정 대응 선택
- 닷컴 버블 붕괴 → 이직/버티기
- 퇴근 후 카카오톡 업무 지시 → 수락/읽씹/거절
- 코로나 거리두기 4단계 → 방역/배달/영업 강행
- AI 인원 재조정 발표 → AI 학습/저항/이직 준비

### 📊 스탯 시스템
- **체력** (0~100): 선택에 따라 소모/회복
- **스트레스** (0~100): 위험 수준 경고 표시
- **멘탈** (0~100): 극단적 상황에서 번아웃

### 💰 재정 시스템
- 시대별 물가 연동 (버스비 50원~1,500원, 짜장면 500원~8,000원)
- 수입/지출 실시간 추적
- 부업 수익 (주식, 코인, 로또, 중고거래, 유튜브)

### 👨‍👩‍👧 가족 관계
- 배우자 친밀도 (0~100): ❤️ 하트로 시각화
- 자녀 친밀도 (0~100): 집안일/대화/놀아주기로 상승
- 관계 악화 시 갈등 이벤트 자동 발생

### 🧹 집안일 퀘스트
- 설거지/빨래/청소기/분리수거/요리/장보기
- 완료/미처리 사이드 패널 표시
- 처리 → 배우자 관계 상승 / 방치 → 관계 하락

### 🏆 엔딩 시스템 (6종)
- 💕 행복한 가정 엔딩
- 🌟 고독한 성공 엔딩
- 💔 이혼 위기 엔딩
- 😵 번아웃 엔딩
- 🙂 평범한 하루 엔딩
- 😔 힘들었던 하루

## 📁 파일 구조

```
index.html          메인 HTML (4개 화면: 타이틀/시대선택/게임/엔딩)
css/
  style.css         전체 스타일 (시대별 CSS 테마 + CRT 효과)
js/
  eras.js           시대 데이터 (6시대 × 경제/직업/거주지/역사이벤트/난이도/프리뷰)
  events.js         하루 이벤트 빌더 (buildDayEvents + 4개 풀 함수)
  game.js           게임 엔진 (캐릭터생성/이벤트진행/사이드패널/엔딩계산)
```
